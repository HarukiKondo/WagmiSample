# WagmiSample
Wagmi + viem を試すためのリポジトリ

### 始め方

```bash
npm create wagmi@latest
```

```bash
cd my-app && yarn && yarn dev
```

### RainbowKit対応済みのプロジェクト始め方

```bash
npm init @rainbow-me/rainbowkit@latest
```

```bash
cd my-rainbowkit-app && yarn && yarn dev
```
